包装类是Java使用极其频繁的类型，他们代表着基本类型的引用形式，并且提供了丰富的方法。

Java数据类型

  • 基本数据类型
  • 引用数据类型

基本数据类型

基本数据类型有8种，分别是byte，short，int，float，double，long，char，boolean。
占用大小和范围请自行搜索。

引用类型

  • 非基本数据类型的对象都是引用类型的，存储在堆中。
  • 引用类型的变量类似于指针，存储的对象的地址。
  • 引用类型的变量的默认值是null。

包装类

Java给提供了每种基本数据类型的包装类，包装类是引用类型的对象，存储在堆中，与基本数据类型的区别是它可以为null，并且相对于基本数据类型提供了很多的方法。

为什么要使用池化技术

  • 运行更快
  • 节省内存
    因为这八种包装类和String类型都是经常使用的，如果代表相同的值而且频繁使用的话，每次都创建新的对象就会十分浪费内存，而且创建对象还会有时间上的开销，所以池化技术减少了创建对象的时间开销，还减少了创建重复对象的内存开销。
